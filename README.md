# 制限時間付き○×クイズアプリ

JavaScriptで作成した、制限時間付きの○×クイズアプリです。  
ボタン操作・タイマー・スコア管理など、Webアプリの基本要素を学ぶ目的で制作しました。

## 機能
- ○×ボタンで回答
- 1問ごとに制限時間（5秒）
- 正解 / 不正解 / 時間切れで表示と効果音を変更
- スコアとランク（S〜C）を表示
- 問題のシャッフル機能
- リスタート機能

## 使用技術
- HTML
- CSS
- JavaScript（Vanilla JS）

## 工夫した点
- `setInterval` と `clearInterval` を使ってタイマー処理を実装した
- クイズの進行状態（現在の問題番号・スコア）を変数で管理した
- 正解・不正解・時間切れで表示スタイルと効果音を切り替え、ゲーム性を高めた
- 関数ごとに役割を分けて、読みやすさを意識した

## 学んだこと
- DOMの取得と操作方法
- イベントリスナーの使い方
- 配列・オブジェクトの扱い
- タイマー処理の仕組み
- Webアプリの基本的な構造
